# Guard::Exec
![gem_version](https://img.shields.io/badge/gem%20version-0.1.0--alpha-red.svg)
## Installation

Add this line to your application's Gemfile:

```ruby
gem 'guard-exec', git: 'https://github.com/clouw/guard-exec'
```

And then execute:

    $ bundle
